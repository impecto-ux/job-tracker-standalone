# Brand System Rules

## 1. Layout Principles
- **Grid:** Use a 12-column grid, but feel free to break it.
- **Micro-alignment:** Elements must either be perfectly aligned or intentionally misaligned by at least 2rem. No "accidental" misalignments.
- **Borders:** All major sections must have visible 1px borders (#333).

## 2. Typography
- **Headings:** uppercase, tight tracking (-0.05em).
- **Body:** Monospace only. No exceptions.
- **Hierarchy:** Size is the only indicator of hierarchy. Avoid multiple font weights (use Bold/Black mostly).

## 3. Interaction
- **Hover:** Instant color inversion (bg <-> fg). No transitions > 0.1s.
- **Cursor:** Default system cursor or custom "crosshair".
- **Scroll:** Native scroll. No smooth-scroll libraries unless necessary for specific effects.

## 4. Components
- **Buttons:** Sharp corners. Uppercase text. Visible border.
- **Images:** Grayscale by default, color on hover.
